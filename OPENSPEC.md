### Typography

**Headings: Fraunces**
- A variable serif with "wonky" optical adjustments
- Feels both authoritative and approachable
- Has organic, slightly imperfect qualities that echo nature

**Body: Source Sans 3**
- Highly readable, professional
- Warm and humanist without being casual
- Excellent accessibility characteristics

**Accent/Quotes: Caveat** (optional)
- Handwritten feel for pull quotes or personal touches
- Adds warmth and humanity

### Visual Language

**Photography Style: Forests & Mountains**
*These landscapes embody the brand: grounded yet aspirational, complex yet navigable*

- **Misty forest paths** â€” Clarity emerging through complexity
- **Mountain peaks at dawn** â€” Vision, aspiration, the long view
- **Aerial forest canopies** â€” Complexity from above, yet organized
- **Forest clearings** â€” Moments of clarity amidst dense information
- **Mountain valleys** â€” Sheltered growth, sustainable ecosystems
- **Ancient trees** â€” Wisdom, stability, deep roots
- **Alpine meadows** â€” Where innovation (peaks) meets foundation (earth)
- **Morning mist over forests** â€” Transparency, revelation, trust

**Suggested Image Sources (Free):**
- Unsplash searches: "misty forest path", "mountain dawn", "aerial forest", "ancient redwoods", "alpine meadow"
- Pexels, Pixabay for additional options

**Iconography:**
- Line-drawn, organic shapes
- Botanical elements (leaves, branches, seeds)
- Subtle, not cartoonish
- Could incorporate circuit/network patterns growing from natural forms

**Texture & Patterns:**
- Subtle paper grain on backgrounds
- Topographic line patterns
- Organic mesh/network visualizations

## 5. Technical Specification

### Recommended Stack: Astro + TailwindCSS

**Why Astro:**
- **Performance:** Ships zero JavaScript by default, incredibly fast
- **Content-focused:** Perfect for a consulting site with mostly static content
- **Flexible:** Can add React/Vue components where needed (contact form, calendar)
- **SEO-optimized:** Server-rendered, excellent Core Web Vitals
- **Easy maintenance:** Markdown for blog posts, simple deployment
- **Modern:** Growing ecosystem, great developer experience

**Core Technologies:**
```
Framework:        Astro 4.x
Styling:          TailwindCSS + custom theme
Animations:       CSS animations + Framer Motion (for interactive elements)
Forms:            Formspree (works great with static hosting)
Calendar:         Cal.com (open-source, self-hostable)
Blog/Insights:    Substack RSS feed integration
Hosting:          GitHub Pages (free, reliable, version-controlled)
Domain:           chandrimaganguly.github.io
CMS (optional):   Decap CMS (formerly Netlify CMS, works with GitHub)
```

**GitHub Pages Configuration:**
```yaml
# .github/workflows/deploy.yml
name: Deploy to GitHub Pages

on:
  push:
    branches: [main]
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
      - run: npm ci
      - run: npm run build
      - uses: actions/upload-pages-artifact@v3
        with:
          path: ./dist

  deploy:
    needs: build
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    steps:
      - uses: actions/deploy-pages@v4
        id: deployment
```
